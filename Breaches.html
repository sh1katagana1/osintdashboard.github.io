    Online Breachdata Resources

    Warning: Be very aware and cautious of your legal standing and agency policy when conducting research on breach and leak data.

    Ripgrep

    • Ripgrep Github Page
    • Linux: sudo snap install ripgrep --classic
    • Mac: brew install ripgrep
    • Windows: choco install ripgrep

    Ripgrep Search Operators

    • Search keyword and store results in text file: rg -a -F -i -N keyword > filename.txt
    • Search 2 words in a line (AND operator) in all files in this folder: rg -i "word1" | rg -i "word2"
    • Search for John or Jon (OR operator) in one file: rg -i "john|jon" facebook.csv
    • Match string ending with '00' add a comma after the last digit (as that's the format in the FB breach): rg '00,\b' america.csv
