    IP Formats - IPv4 vs IPv6

    IPv4

    Example: 97.74.104.15

    Most addresses are still IPv4 which are easier to deal with but are more apt to be “dynamic” which means that they might only be associated with a target for a limited period time.

    IPv6

    Example: 2607:fb90:206f:c64c:f907:c741:274b:45d5
